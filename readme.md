## BUYSKET

### Introdução
Este repositório contém o código para utilizar o display QRCODE com o microcontrolador ESP. O display utiliza as seguintes portas:

| Display QRCODE | ESP        |
| -------------- | ----------|
| VCC            | 3.3v       |
| GND            | GND        |
| CS             | D1         |
| RESET          | RST        |
| D/C            | D4         |
| SDI(MOSI)      | D7         |
| SCK            | D5         |
| LED            | 3.3V       |
